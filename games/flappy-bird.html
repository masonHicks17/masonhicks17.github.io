Flappy Cave

Guide your bat through a dark underground cave. Dodge the stalactites and stalagmites!

Score: 0
Click / Tap / Space to flap

About This Game

A redesign of the classic Flappy Bird with a cave theme — stalactites hang from above, stalagmites rise from below. Built to practice Canvas rendering, game state machines, and physics simulation.

  • Countdown start screen so you're never caught off guard
  • Smoother, more forgiving physics than the original
  • Procedural cave rock formations
  • High score tracked per session

Technical Details

Tech: Vanilla JavaScript, HTML5 Canvas API

Pattern: Game state machine — start → countdown → playing → dead

Physics: Gravity 0.22, jump impulse −5.5, pipe speed 2px/frame
