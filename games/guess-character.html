Guess the Disney Character ✨

Think of any Disney character. I'll ask you yes/no questions and try to figure out who it is.

How It Works

The game uses an information-gain algorithm — each question is chosen to split the remaining candidates as evenly as possible, similar to a binary search. Answer Yes, No, Maybe, or I Don't Know. Maybe/IDK keep all candidates in play. The game covers 32 Disney characters across classic and modern films.
