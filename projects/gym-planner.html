Gym Planner Assistant

Next.js AI Integration Fitness

Overview

An AI-powered workout planning tool that generates personalized training routines based on your fitness goals, available equipment, and time constraints. The app uses Claude to create smart, adaptable routines that evolve with your progress.

The Problem

Generic workout plans don't account for individual constraints. I wanted a tool that would ask the right questions and generate routines tailored to my specific situation—not just another one-size-fits-all program.

The Solution

I built a conversational interface that gathers your requirements (goals, equipment, time availability) and uses AI to generate customized workout plans. Users can refine the plan by providing feedback, and the AI adapts accordingly.

Key Features

  • Conversational onboarding flow
  • AI-generated personalized workout routines
  • Exercise demonstrations and form tips
  • Progress tracking and plan adjustments
  • Export routines as PDF
  • Mobile-friendly interface

Technical Stack

Frontend: Next.js, React, TypeScript, Tailwind CSS

AI: Claude API, prompt engineering

Database: Supabase (PostgreSQL)

Deployment: Vercel

AI Workflow Insight

This project was a great exercise in prompt engineering. I learned how to structure prompts for consistency, handle edge cases gracefully, and use the API efficiently. The key was understanding how to break the problem into smaller, atomic prompts.
