Personal Dashboard

React Data Viz Personal

Overview

A unified personal dashboard built to consolidate my daily metrics, habits, and goals in one place. The dashboard provides quick insight into what matters most—health, learning, and creative output.

The Problem

I was tracking metrics across multiple apps and spreadsheets. It was fragmented and I wasn't getting the full picture. I wanted a single source of truth that was beautiful to look at and actually useful for reflection.

The Solution

I built a dashboard that pulls data from various sources and presents it in a clean, at-a-glance format. The design prioritizes clarity and uses data visualization to make patterns obvious. Each metric card is interactive and provides deeper insights on click.

Key Features

  • Real-time data aggregation from multiple sources
  • Custom data visualizations with Chart.js
  • Interactive metric cards with drill-down capabilities
  • Responsive design for desktop and mobile
  • Dark mode support for late-night tracking
  • Data export functionality (CSV/JSON)

Technical Stack

Frontend: React, TypeScript, Tailwind CSS, Chart.js

Backend: Node.js, Express, MongoDB

Tools: Figma, Git, Vercel

What I Learned

This project taught me a lot about data visualization and UX design. I learned how to make complex information accessible and beautiful. I also discovered the importance of user feedback—my first version was pretty but not practical.
