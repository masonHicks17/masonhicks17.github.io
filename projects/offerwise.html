Overview

OfferWise is a practical job offer evaluator that turns a messy career decision into a scored, comparable summary. It rates compensation, benefits, work-life balance, growth potential, and company culture so the user can see the true value of an offer instead of reacting only to salary.

The Problem

Job offers are hard to compare because the most important tradeoffs are spread across salary, benefits, role fit, risk, flexibility, and future upside. A high salary can hide weak benefits or a poor growth path, while a lower salary can still be a better long-term offer.

The Solution

I built a guided scoring tool that collects key offer details, translates them into category ratings, and produces an overall score with targeted improvement notes. The experience is designed to feel fast and readable, with a mobile-first interface and clear category-level feedback.

Key Features

  • Guided offer input flow for compensation, benefits, culture, growth, and balance
  • Weighted category scoring that rolls up into a simple 10-point overall rating
  • Market-aware compensation checks using benchmark data and location adjustments
  • Recommendation logic that highlights where the offer can improve
  • Responsive interface with a phone-friendly layout and desktop-friendly review screen

Technical Stack

Frontend: HTML, CSS, and vanilla JavaScript with modular UI rendering

Scoring: Category scoring modules for compensation, market comparison, recommendations, and labels

Data: Local benchmark and market data files for deterministic, inspectable scoring

Testing: Scoring tests covering offer evaluation logic and label outputs

What I Learned

The useful part of a scoring tool is not just the score. It is the explanation behind the score. OfferWise pushed me to separate input collection, scoring logic, market comparison, and recommendation copy so the tool feels transparent instead of arbitrary.
