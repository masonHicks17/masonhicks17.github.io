Landing Page Collection

Design Frontend Collection

Overview

A curated collection of premium landing pages exploring modern design patterns, micro-interactions, and conversion optimization techniques. Each page is a focused study in visual design and user experience.

Purpose

This collection serves as both a design exploration and a reference library. I built these pages to understand conversion psychology, test animation patterns, and develop reusable component systems. Each page focuses on a specific design principle.

Pages Included

  • SaaS Landing: Enterprise-focused design with clear value proposition
  • Product Launch: High-energy design with interactive elements
  • Minimal Pitch: Typography-forward design with restraint
  • Case Study Hub: Content-rich layout with elegant typography
  • Pricing Page: Conversion-optimized comparison interface

Technical Stack

Frontend: HTML, CSS, JavaScript

Animation: CSS animations, Intersection Observer API

Design Tools: Figma

Design Learnings

Building this collection taught me about the relationship between design and conversion. I learned that sometimes less is more, that whitespace is powerful, and that animations should serve a purpose—not just delight for delight's sake.
