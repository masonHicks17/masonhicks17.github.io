Flappy Bird Clone

A clean remake of the classic reflex game. Built to explore game mechanics and canvas rendering.

Score: 0

How to Play:

Click or tap to make the bird jump. Avoid the pipes!

About This Game

This is a clean, educational remake of Flappy Bird. I built it to understand:

  • Canvas API and drawing operations
  • Game loop implementation
  • Collision detection
  • Simple physics simulation

Technical Details

Tech: Vanilla JavaScript, HTML5 Canvas

Size: ~3KB (unminified)

Browser Support: All modern browsers
